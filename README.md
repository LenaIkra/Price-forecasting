# Price Forecasting

Проект для прогнозирования цен розничных товаров на основе данных M5 Forecasting.

## Цель проекта

Сравнить несколько подходов к прогнозированию цен:

- классические модели временных рядов;
- модели машинного обучения;
- нейросетевые модели;
- гибридные модели.

Целевая переменная: `sell_price`.

## Используемые модели

- SARIMA
- Gradient Boosting
- M5-style модель
- LSTM
- LSTM с эмбеддингами категориальных признаков
- Hybrid SARIMA + Boosting

# Данные

В проекте используются файлы M5 Forecasting:
sales_train_evaluation.csv
calendar.csv
sell_prices.csv

Файлы должны быть расположены в директории:
data/raw/

Установка зависимостей
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt

Запуск основного эксперимента
python3 main.py

Запуск финального эксперимента
python3 main_final.py

Результаты
reports/final_run/final_comparison.csv
reports/final_run/final_comparison_sorted.csv

Метрики отдельных моделей сохраняются в подпапках:
reports/final_run/sarima/
reports/final_run/boosting/
reports/final_run/m5_style/
reports/final_run/hybrid_sarima_boosting/

Метрики качества
Для оценки моделей используются:
	•	MAE
	•	RMSE
	•	WAPE

Особенности реализации
	•	разбиение train/test выполняется по времени;
	•	для SARIMA используется глобальный подбор параметров на подмножестве рядов;
	•	для табличных моделей используются лаговые, скользящие и календарные признаки;
	•	для контроля потребления памяти табличные модели в финальном прогоне обучаются на подмножестве наиболее длинных рядов;
	•	результаты сохраняются в CSV и JSON-файлы.

Основной вывод

В ходе экспериментов сравниваются классические временные модели, модели машинного обучения и гибридные подходы. Финальный выбор модели выполняется по значениям метрик на тестовой выборке.
