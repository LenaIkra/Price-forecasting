# Прогнозирование цен потребительского ритейла с использованием методов машинного обучения

Репозиторий содержит программную реализацию выпускной квалификационной работы по теме:

**«Прогнозирование цен потребительского ритейла по тестовой выборке с использованием методов машинного обучения»**

Проект посвящен исследованию методов прогнозирования цен в потребительском ритейле на основе временных рядов, данных о продажах, ценах и календарных событиях.

---

## Цель проекта

Цель работы — разработка модели прогнозирования цен в потребительском ритейле на основе методов машинного обучения и анализа временных рядов.

В рамках проекта были выполнены:

- анализ существующих подходов к прогнозированию цен;
- подготовка и исследование набора данных M5 Forecasting;
- формирование временных, лаговых и календарных признаков;
- обучение и сравнение моделей прогнозирования;
- независимая валидация лучшей модели;
- оценка потенциального экономического эффекта.

---

## Используемый набор данных

В работе использовался набор данных **M5 Forecasting Dataset**, содержащий информацию о продажах товаров, ценах и календарных событиях сети Walmart за период 2011–2016 гг.

В рамках исследования данные были преобразованы в аналитический панельный датасет, включающий:

- данные о продажах товаров;
- цены товаров;
- календарные события;
- признаки магазинов и товарных категорий;
- лаговые признаки;
- признаки скользящих средних.

Исходные данные не включены в репозиторий из-за большого объема файлов.

## Исследуемые модели

В работе были реализованы и сравнены модели разных классов:

| Модель | Тип подхода |
|---|---|
| SARIMA | статистическая модель временных рядов |
| Gradient Boosting | ансамблевая модель машинного обучения |
| M5-style Boosting | модификация boosting-подхода |
| LSTM + Embeddings | нейросетевая модель |
| Hybrid Boosting + M5 | гибридная ансамблевая модель |

---

## Метрики качества

Для оценки качества прогнозирования использовались следующие метрики:

- **RMSE** — корень из среднеквадратичной ошибки;
- **MAE** — средняя абсолютная ошибка;
- **WAPE** — взвешенная абсолютная процентная ошибка.

Основной метрикой сравнения моделей была выбрана **RMSE**, поскольку она сильнее штрафует крупные ошибки прогнозирования.

---

## Результаты эксперимента

| Модель | RMSE | MAE | WAPE |
|---|---:|---:|---:|
| Gradient Boosting | **0.0531** | **0.0137** | **0.0032** |
| Hybrid Boosting + M5 | 0.0535 | 0.0138 | 0.0032 |
| M5-style Boosting | 0.0578 | 0.0144 | 0.0034 |
| LSTM + Embeddings | 1.4570 | 1.2603 | 0.2959 |
| SARIMA | 5.44 × 10^62 | 2.28 × 10^60 | 5.35 × 10^59 |

Наилучшее качество прогнозирования показала модель **Gradient Boosting**.

## Независимая валидация

Для проверки устойчивости лучшей модели была проведена независимая валидация на временных рядах, не использовавшихся в основном вычислительном эксперименте.



Для запуска независимой проверки модели использовался модуль:

```bash
python3 -m src.independent_validation
```

В рамках независимой валидации выполнялись:

- загрузка обученной модели;
- прогнозирование цен на новых данных;
- расчет метрик качества;
- bootstrap-оценка устойчивости результатов;
- расчет доверительных интервалов метрик качества.

Результаты независимой проверки сохранялись в директории:


Для оценки потенциального экономического эффекта использовался модуль:


В рамках расчета экономического эффекта выполнялись:

- анализ сценариев недооценки и переоценки цены моделью;


| Суммарное абсолютное отклонение выручки | 6 979.04 |
